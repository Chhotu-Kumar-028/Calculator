# Web Calculator 🧮

**[🚀 View Live Demo](https://calculator-gray-chi-37.vercel.app/)**

A sleek and interactive web-based calculator built with HTML, CSS, and JavaScript. This project was developed to demonstrate fundamental frontend development skills, focusing on functional logic, UI design, and clean code practices.

## 🚀 Features
- **Basic Arithmetic Operations**: Supports addition, subtraction, multiplication, and division.
- **Clear & Delete Functionality**: Includes an `AC` (All Clear) button to reset the display and a `DE` (Delete) button to remove the last entered character.
- **Secure Evaluation**: Utilizes `new Function()` instead of the standard `eval()` to perform calculations securely, demonstrating awareness of safe coding practices.
- **Modern UI**: Features a clean, neumorphic-inspired design with soft shadows and a visually appealing color palette.

## 🛠️ Technologies Used
- **HTML5**: For structuring the calculator layout and form elements.
- **CSS3**: For styling the interface, utilizing flexbox for alignment and custom shadows for depth.
- **JavaScript (ES6+)**: For implementing the calculator logic, handling DOM events, and processing mathematical expressions safely.

## 💻 Getting Started

To run this project locally, simply follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Chhotu-Kumar-028/calculator-project.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd calculator-project
   ```
3. **Open the application:**
   Open the `index.html` file in any modern web browser to use the calculator.
